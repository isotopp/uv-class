# Python Project Management with UV



---

This repository contains a hugo-based website for documentation and project information, using submodules for themes.

To check this out, do one of either:

1. `git clone --recurse-submodules https://github.com/isotopp/uv-class.git`
2. `git clone https://github.com/isotopp/uv-class.git && cd uv-class && git submodule update --init --recursive`
